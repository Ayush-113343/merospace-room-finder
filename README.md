# MeroSpace - Room Finder

## Project Overview

MeroSpace is a web-based room rental finder application that connects room owners with tenants. The platform allows room owners to advertise available rental rooms while enabling tenants to search, compare, and contact room owners easily.

---

## Problem Statement

Finding rental rooms is often difficult because users rely on social media, friends, or local advertisements. This process is time-consuming and frequently contains outdated or inaccurate information.

MeroSpace provides a centralized platform to simplify room searching and room advertisement.

---

## Project Objectives

- Help tenants find rental rooms quickly.
- Help room owners advertise available rooms.
- Reduce the time required to search for rental rooms.
- Provide a simple and user-friendly platform.
